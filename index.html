LEO GitHub 文件加速

主域名 cf.leo123.cn，建议收藏。

支持 GitHub Releases、archive、gist、raw 等文件加速下载

支持带或不带协议头的 GitHub 链接，release/archive 通过 Cloudflare 加速。

raw 文件将跳转至 JsDelivr CDN。暂不支持项目文件夹。

示例：
https://cf.leo123.cn/https://raw.githubusercontent.com/XXX/XXX.json

✅ 合法输入示例：

分支源码：https://github.com/LEO/project/archive/master.zip

release 文件：https://github.com/LEO/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/LEO/project/blob/master/filename

项目由 LEO 部署，开源自 GitHub。
