LEO GitHub 文件加速

主域名 cf.leo123.cn，请访问和收藏本站

GitHub文件、Releases、archive、gist等代理加速下载服务

使用说明

• GitHub文件链接带不带协议头都可以，支持release、archive以及文件

• release、archive使用Cloudflare加速，文件会跳转至JsDelivr

• 注意：不支持项目文件夹

• 示例：https://cf.leo123.cn/https://raw.githubusercontent.com/XXX/XXX.json

合法输入案例

分支源码：https://github.com/LEO/project/archive/master.zip

release源码：https://github.com/LEO/project/archive/v0.1.0.tar.gz

release文件：https://github.com/LEO/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/LEO/project/blob/master/filename
